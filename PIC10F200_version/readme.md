desperate firefly PIC10F200 version
====================================

The PIC10F200 is the simplest MCU available on market

1 8 bit TIMER

1 Wathdog timer

3 I/O

1 ~RST/input pin

256 bytes FLASH

16 bytes RAM

That's all there in it.

schematic
---------
                                  PIC10F200
				+-------------+
			      1-|N/C      ~RST|- 8 
		          +3v-2-|Vdd       Vss|- 7-GND  
                  +3v-LED-1K -3-|GP2       N/C|- 6 
	                      4-|GP1       GP0|- 5
				+-------------+
